# F1 ANALYTICS FOR BEGINNERS
A project designed for newbies that wanted to explore the data around F1



## 🎯 Purpose
This project is created to make F1 data more accessible for beginners in F1, or just wanted to know more about data and analytics.
This project simplifies race strategies, lap times, driver performance by using dashboards and simple explainations/

## 🚀 Features
- 📊 Simple race data visualizations
- 🏎️ Driver performance comparison
- 📈 Lap time analysis
- 🤖 (Planned) AI insights 

## 🛠️ Tech Stack
- Python
- FastF1.com API
- Flask
- HTML/CSS
- JavaScript (later)
